# GPU Host Copilot 공개 베타

전기료와 실제 작업 시간을 반영해 GPU 호스팅 순이익을 계산하는 정적 로컬 우선 도구입니다.

공개 베타: https://polykarpo.github.io/gpu-host-copilot/

## 개인정보

계산기 자체에는 분석 도구, 광고 추적기, 로그인, 쿠키, 브라우저 저장소, 백엔드 제출 기능이 없습니다. 입력값은 현재 탭에서만 계산됩니다. GitHub Pages는 보안을 위해 방문자 IP 주소를 기록할 수 있습니다. 자세한 내용은 사이트의 개인정보 안내를 확인하세요.

## 한계와 독립성

- 하드웨어 감가상각, 세금, 냉각·대기 전력, 환전 수수료는 계산에 포함되지 않습니다.
- 결과는 예측이 아니며 수익이나 작업 배정을 보장하지 않습니다.
- 이 프로젝트는 공개 베타이며 Salad Technologies와 무관한 독립 비제휴 도구입니다.
